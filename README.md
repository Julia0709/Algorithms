# Algorithms

To make a new directory, enter

```
$ gulp init
```

**mode:**
Choose a mode

| N   | Directory   |
| :-- | :---------- |
| 0   | Leetcode    |
| 1   | Hacker Rank |
| 2   | Others      |

```
prompt: mode:  0
```

**name:**
enter the question title (directory name)

```
prompt: name: [Qurestion title]
```
